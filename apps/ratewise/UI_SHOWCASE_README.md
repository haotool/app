# UI Showcase 頁面使用指南

## 📍 訪問路徑

開發環境：`http://localhost:5173/ui-showcase`
生產環境：`https://app.haotool.org/ratewise/ui-showcase`

## ✨ 功能特點

### 配色方案切換

頁面支援三種配色方案的即時切換：

1. **💙 RateWise 品牌配色**（當前使用）
   - 藍紫漸變品牌色
   - 專業可信賴的形象
   - 圓潤現代的視覺元素

2. **🍬 棉花糖甜心配色**
   - 粉紫粉藍漸變色調
   - 可愛甜美的風格
   - 多彩和諧的配色

3. **☁️ 粉彩雲朵配色**
   - 柔和天空色調
   - 清晨薄霧的柔和感受
   - 純淨清新的配色

### 組件狀態展示

1. **離線模式已就緒** (✨)
   - 當 PWA 首次安裝完成後顯示的狀態
   - 綠色勾選圖標
   - 單按鈕操作（好的）

2. **發現新版本** (🎉)
   - 當有新版本可用時顯示的狀態
   - 藍色刷新圖標
   - 雙按鈕操作（馬上更新 / 等等再說）

### 互動測試

頁面提供三個控制按鈕：

- **顯示離線模式** - 在右上角展示離線模式提示
- **顯示更新提示** - 在右上角展示更新提示
- **關閉** - 關閉當前顯示的提示

### 配色系統

根據選擇的配色方案，動態展示對應的色卡：

**RateWise 品牌配色：**

- 主要漸變：blue-500 → indigo-600
- 背景漸變：blue-50 → indigo-50 → purple-50
- 邊框色：blue-200
- 標題文字：blue-900
- 描述文字：indigo-700
- 按鈕背景：blue-600

**棉花糖甜心配色：**

- 圖標漸變：pink-200 → purple-200 → blue-200
- 背景漸變：pink-50 → purple-50 → blue-50
- 邊框色：purple-100
- 標題文字：purple-700
- 描述文字：purple-500
- 按鈕漸變：pink-300 → purple-300 → blue-300

**粉彩雲朵配色：**

- 圖標漸變：purple-200 → blue-200
- 背景漸變：purple-50 → blue-50 → purple-100
- 邊框色：purple-200/50（半透明）
- 標題文字：purple-800
- 描述文字：purple-600
- 按鈕漸變：purple-400 → blue-400

### 設計特點

詳細說明組件的六大設計特點：

1. 藍紫漸變品牌色
2. 圓潤現代視覺（24px 圓角）
3. 柔和光暈效果
4. Emoji 點綴
5. 彈性入場動畫
6. 完整無障礙支援
