# 星噗噗 StarPuff 全關卡攻略（L1–L20）

> 對應遊戲版本：`@app/starpuff` 0.13.0（v14 UX/PWA 列車）。
> 本攻略之路線與打法全數經真實瀏覽器實測驗證（Playwright 真機遊玩，非純資料推導）；
> 截圖存放於 QA 產物目錄 `screenshots/starpuff-walkthrough/`（repo 忽略清單內，僅供稽核佐證）。
> 更新約定：關卡資料 SSOT（`src/game/logic/levels.ts`）或魔王 FSM 變更時，本文件必須同 PR 更新對應章節。

## 通用操作與核心循環

- 行動裝置：左半屏浮動搖桿移動；右側 A 鍵跳躍（空中連按拍翅最多 3 次）、B 鍵吸／射。
  桌機備援：方向鍵移動、Z 跳躍、X 吸射。
- 核心循環：長按吸入鍵吸入小怪 → 吞下得星彈（上限 3 發，吞什麼味射什麼味）→ 點按發射。
  **吞入與射殺同樣計入擊殺配額**——搶配額時直接吞最快。
- 下衝擊（下砸）：真空中「搖桿下＋跳躍鍵」觸發，落地 60px 衝擊波 2 傷、冷卻 1.2 秒、零彈藥消耗、可破磚。
  零彈藥時是唯一可靠的攻擊手段，記住它。
- 連吞同種怪 ×2 成強化星（傷害 ×1.6）；連吞兩種不同怪可合成混合星（見各關「星味」節）；
  滿匣長按 B 0.8 秒放星暴大招（全場清怪＋魔王 12 傷＋5 秒無敵）。
- 同系星彈滿 3 發時，地面長按吸入鍵 0.6 秒星化變身（雷化／風化／殼化，10 秒）；
  全部變身與混合星皆為加成路線，**任何關卡純標準星彈都能通關**（官方 anti-softlock 保證）。
- 走動關死亡重試當前關（配額歸零重算）；卡點關（L11／L15／L19）越過中點後死亡自 checkpoint 就地重生，
  配額與計時保留。魔王關死亡進敗北結算，按重試自前室重來。
- 慈悲補血：血量剩 1 顆心且久戰時，場上會隨機出現愛心（每命最多 2 次）——瀕死時留意撿取。

## 精英房通則（走動關中段）

每個走動關中段有一隻精英變體（放大、變色、血量池），玩家靠近時武裝：生成精英＋粉色軟鎖門擋路。

- 精英不可吸入，只能星彈或下砸磨死；擊敗開門，掉落稀有星味小怪＋回復愛心（+2 HP）。
- 打法通則：站在房界外緣（精英出不了房），等精英走到房界邊緣時遠距點射——零風險磨死。
- 保底：60 秒逾時門自動開，精英留場可直接跑過（不擊敗亦不影響過關）。
- 星星門已開（配額達成）後精英房不再武裝。

## 第一區：果凍平原（L1–L4）

### L1 果凍草原

- **主題**：教學關——移動／吸入／發射閉環，單向平台首見。世界寬 2700，配額 6，幾乎無死亡壓力。
- **路線**：開局先「反向」走到世界最左緣拿彩蛋（見下），再一路右行。地面平坦，
  平台三座（x≈700／1400／2050）皆為支線，主線走地面即可。擊殺（含吞入）滿 6 隻後右端星星門開啟，走入過關。
- **怪物**：jelly 果凍丁（60%，跳向玩家，可吸→標準星）、floaty 飄飄鳥（40%，正弦飄移，可吸→疾風星）。
  全部可吸——見怪就吞，配額很快就滿。
- **精英**：粉紅暴走果凍丁（x≈1500，HP 10，跳頻 1.5 倍）。房外點射兩發星彈即倒；
  掉落 glowy 流光味小怪＋回復愛心。
- **彩蛋**：`reach-x`——開局反向走到最左緣（x<60）觸發，+1 HP（實測觸發於開場 1 秒內，穩定）。
- **實測**：59.6 秒通關、零死亡（截圖 `l01-start.png`、`l01-elite.png`、`l01-clear.png`）。

### L2 雲朵高台

- **主題**：垂直平台閱讀＋不可吸威脅初登場。世界寬 3100，配額 9。移動平台與彈簧墊首見。
- **路線**：主線走地面雙層。x≈1150 有彈簧墊（踩上彈飛，可上 272 高層雲台）；
  x≈2620 移動平台（水平來回）。配額滿後右端進星星門。
- **怪物**：floaty 30%、spiky 刺刺瓜 25%（地面滾衝，**不可吸**，吸會被彈開）、puffy 氣球魨 20%
  （高空緩降，**吸入安全**；用星彈打死會爆 4 向刺彈，遠距 260px 以上再射）、
  shelly 殼殼 15%（僅暈眩窗可吸）、glowy 提燈水母 10%（週期釋放範圍脈衝，預警圈擴張時遠離，可吸→流光星）。
- **應對心法**：spiky／shelly 用星彈遠距點殺或下砸；puffy 優先吸（不爆），其次遠射；
  彈藥永遠留 1 發應付滾來的 spiky。
- **精英**：鋼青重殼殼（x≈1700，HP 14，走速 1.4 倍）。房界外緣（x≈1950 門、房左界約 1400）站定，
  等它撞到房界時點射 3 發；掉 drilly 重鑽味。
- **彩蛋**：`stand-count`——最高雲朵平台（272 高層）連續站上 3 次，獎勵瞬間滿彈匣。
  彈簧墊起跳最容易達成。
- **實測**：119.5 秒通關、零死亡（截圖 `l02-start.png`、`l02-elite.png`、`l02-clear.png`）。
  實測備註：首遇 puffy 會跳出「爆裂星：命中小範圍爆炸」提示浮字。

### L3 星空回廊

- **主題**：八種怪高壓混編＋彈藥管理；可破壞磚首見。世界寬 3500，配額 10。
- **路線**：主線地面推進；可破壞磚 ×3（x≈550 彈藥／1950 回血／3050 彈藥）立於地面支線，
  下砸或星彈打破拿補給，單跳亦可直接越過。x≈1600 彈簧、x≈2380 垂直移動平台。
- **怪物**：jelly／floaty／puffy／glowy 照舊；新增 chompy 咬咬花（鵝黃花，定點紮根，**不可吸**、HP 10，
  玩家進 120px 前搖 0.4 秒後咬合——遠距兩發星彈或直接跳過）、zappy（可吸→雷鏈星，命中跳電清群）、
  drilly 鑽鑽鼴（潛地期免傷不可吸，破土 1.4 秒窗才能吸／打；吞下得重鑽星）、spiky。
- **應對心法**：這關的死亡點是 chompy 卡口＋彈藥斷檔。吞 zappy 得雷鏈星一發清簇最划算；
  drilly 潛地時別浪費子彈，等破土窗。
- **精英**：暗紫狂咬花（x≈1900，HP 20，攻速 1.6 倍）。定點紮根不會追擊，
  站房界外 350px 內遠距射 4 發即倒；掉 glowy。
- **彩蛋**：`eat-sequence`——依序連吞 jelly→floaty→puffy，獎勵金星彈一發（傷害 20，留給精英或 chompy）。
- **實測**：140.7 秒通關、零死亡（截圖 `l03-start.png`、`l03-elite.png`、`l03-gate.png`）。
- **重要警示（實測發現）**：長時間拖關不殺紮根怪（chompy 群＋精英）可能把同屏怪數上限（5）佔滿，
  導致生成器停轉、場上無可吸怪且彈藥歸零——此時唯一自救是下砸（零彈藥）清掉紮根怪重開補給循環。
  詳見文末「實測發現問題」第 1 條。

### L4 魔王城（魔王：果凍王 Jellord）

- **主題**：第一魔王——地面型三階段。HP 60、體傷 1。前室補給廊道體系首見。
- **前室（400px 廊道）**：入場即有慈悲愛心（+2 HP）＋可吸補給怪 3 隻（進場先吞滿 3 發彈匣）
  ＋增益二選一台座：**護盾泡**（吸收 1 次任意傷害）／**星力果**（星彈傷害 ×1.5，10 秒）。
  越過前室右緣果凍門柱即鎖閉，死亡重試自前室起。
  建議：首戰求穩選護盾泡；熟練後選星力果加速 P1。
- **三階段招式與反制**：

| 階段 | 門檻     | 招式循環                                                                 | 反制                                                                                   |
| ---- | -------- | ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------- |
| P1   | HP > 50% | idle(1.2s) → 果凍雨（5 顆拋物線彈）→ idle → 震地（跳起落地雙向地面震波） | 果凍雨看落點左右小步走位；震波貼近時跳過；idle 窗即射擊窗                              |
| P2   | HP ≤ 50% | 變紅憤怒、速度 ×1.3；果凍雨升 7 顆；新增衝刺（水平衝撞、撞牆回彈）       | 衝刺沿地面來——起跳從頭頂讓過；此階段場中高風險位會刷疾風靴（移速 ×1.3，10 秒未拾淡逝） |
| P3   | HP ≤ 25% | 果凍雨帶追蹤、震地升級全場震落（quake）                                  | 全場震落靠拍翅滯空迴避；追蹤雨用拉開距離讓彈自然落地                                   |

- **保底打法（純標準星）**：吸前室與補生小怪（每損 10 HP 掉補給、彈藥飢荒立即補生）→
  站魔王 400px 外遠帶點射 → 只用跳躍閃震波與衝刺即可磨死（12 發標準星）。實測 47 秒擊破。
- **進階路線（非必需）**：① 補給含 shelly——暈眩窗吸入湊 3 發殼盾星可星化殼化，
  站定反彈果凍雨（每顆反傷 3）；② 頭頂下砸命中觸發 0.9 秒短暈輸出窗（命中回彈不吃體傷）；
  ③ 滿彈匣長按 B 放星暴（12 傷＋5 秒無敵）速刪血條。
- **彩蛋**：`crown-early-hit`——魔王可受擊後 5 秒內命中皇冠，+1 HP（實測穩定觸發：
  開戰立即滿彈連射即可）。
- **實測**：47.1 秒擊破、重試 1 次（截圖 `l04-start.png`、`l04-arena.png`、`l04-clear.png`）。

## 第二區：天風峽域（L5–L7）

### L5 翔風峽谷

- **主題**：上升氣流柱垂直分層探索。世界寬 3300，配額 10。
- **路線**：主線走地面雙層即可。氣流柱 ×3（x≈1000／1900／2600，恆常供力）
  ——踩進柱域被托升至柱頂（150／150／170 高），可達 208 高台（x≈1000／1900 兩座，
  氣流柱專屬支線）。x≈1550 彈簧。配額滿後右端進星星門。
- **怪物**：gusty 風飄鳥 30%（drift 期有側風推離、玩家在斜下方時俯衝——**俯衝與回升時最貼地，
  是吸入窗**；可吸→疾風味）、spora 孢子菇 20%（定點紮根，週期向頭頂噴孢子雲滯留 1.6 秒
  ——別站牠頭上；恆可吸→孢子星：命中緩速＋持續傷）、spiky 20%、floaty 15%、jelly 15%。
- **應對心法**：gusty 在高處別追（側風會把你推開），等牠俯衝落地窗口吸；
  spora 紮根不動，走近前先確認脈衝預警圈——**牠會佔同屏怪額度，見到就順手吞掉**。
- **精英**：狂風飄鳥（x≈1650，HP 16，俯衝側風 1.4 倍）。飛行精英彈道難抓，
  等牠俯衝進地面帶再射；掉 boomy 迴旋味（L6 主場怪先行體驗）。打不動就等 60 秒門開直接過。
- **彩蛋**：`stand-count`——乘氣流連續兩次站上 208 高台，+1 HP。
- **實測**：336.3 秒通關、死亡 3（截圖 `l05-start.png`、`l05-elite.png`、`l05-gate.png`）。
  實測備註：本關可吸怪供給偏緊（gusty 高飄＋spora 遠處紮根時，地面帶常無彈可補），
  彈藥斷檔時間明顯長於前四關——見文末「實測發現問題」第 2 條。

### L6 迴聲石廊

- **主題**：全怪種混編高壓（十二種同場）＋雙精英。世界寬 3600，配額 12。
- **路線**：主線地面推進。雙移動平台（x≈1450 水平／2900 垂直）、雙彈簧（x≈1900／3200）、
  破磚 ×2（x≈600 彈藥／2050 回血）。**雙精英房兩段**（前段 x≈1300、後段 x≈2500，房距 1200
  不重疊）——一路上會被攔兩次。
- **怪物**：boomy 迴力殼 16%（投擲迴旋殼刃，去程折返雙程都有判定——看到舉殼定身就跳；
  恆可吸→迴旋星）為主場，其餘 spora／gusty／spiky／jelly／floaty／puffy／chompy／zappy／
  shelly／drilly／glowy 全數登場。
- **應對心法**：混編場合彈藥永遠不夠——優先吞 zappy（雷鏈星跳電清群）與 boomy（迴旋星雙程判定）。
  chompy 與 spiky 交給下砸或繞過。
- **精英**：①重殼迴力守衛（boomy 基底，x≈1300，HP 16，掉 glowy 流光味）；
  ②暗雷水母（zappy 基底，x≈2500，HP 18，追速 1.5 倍，掉 drilly 重鑽味）。
  兩房都適用房外風箏射殺；第二隻會主動追擊，保持 200px 以上距離邊退邊射。
- **彩蛋**：`eat-sequence`——依序連吞 boomy→zappy，金星彈一發（實測觸發成功；
  這也是「電鋸迴旋」混合星配方預告：boomy+zappy 混合＝迴旋雙程＋每次命中跳電 ×2）。
- **實測**：304.3 秒通關、死亡 3（截圖 `l06-start.png`、`l06-elite.png`、`l06-gate.png`）。

### L7 蝕月王座（魔王：暗月蝠王 Noctra）

- **主題**：第二魔王——空中型三階段。HP 52、體傷 1。盤旋帶已降至單跳星彈可打高度，
  且一般星彈對魔王帶微導向（地面平射會自然上彎入盤旋帶）——站地上點射就有效。
- **前室**：慈悲愛心＋補給 3 隻＋二選一：**護盾泡**／**疾風靴**。P2 場中高風險位刷星力果。
  建議：護盾泡（空戰彈幕多，擋一次落彈很值）。
- **三階段招式與反制**：

| 階段 | 門檻     | 招式循環                                                                               | 反制                                                                                 |
| ---- | -------- | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| P1   | HP > 60% | 盤旋投彈 ×2（落點預警圈）→ 俯衝（變色前搖 0.72 秒 → 鎖定點撲擊 → 落地滯留 0.3 秒）     | 投彈看預警圈側移；俯衝鎖定後不修正——前搖時往旁走兩步即空振；**落地滯留＝最大輸出窗** |
| P2   | HP ≤ 60% | 投彈 ×3 → 俯衝 → 召喚 floaty（場上限 2）                                               | 召喚的 floaty 順手吞掉補彈；雷化形態的鏈電命中可直接中斷召喚                         |
| P3   | HP ≤ 30% | 狂暴彈幕（放射彈環 ×7）→ 全場俯掠（橫帶閃爍預警 0.6 秒 → 貼 280 高度帶速掠全場）→ 投彈 | 彈環站遠等縫隙；**俯掠帶在頭頂高度——站著不跳就打不到你**，千萬別起跳                 |

- **保底打法（純標準星）**：補給一次只生一隻（間隔 4.5 秒），吞了就射、彈藥飢荒會立即補生。
  站牆側錨點讀招，俯衝滯留窗與投彈間隙輸出，40-60 秒可磨死。
- **進階路線（非必需）**：① 俯衝落地 0.6 秒窗內頭頂下砸→長暈 1.8 秒；② 場內雙彈簧板
  （x≈190／664）彈射到空中貼打盤旋帶；③ 補給含 zappy——雷化鏈電斷召喚；④ 風化自由飛行貼身輸出。
- **彩蛋**：`crown-early-hit`——可受擊後 5 秒內首次命中，+1 HP（實測穩定觸發）。
- **實測**：25.8 秒擊破、零死亡零重試（截圖 `l07-start.png`、`l07-arena.png`、`l07-p2.png`、`l07-p3.png`）。

## 第三區：幻晶深域（L8–L12）

### L8 磁極洞窟

- **主題**：磁力域干擾彈道＋地下突襲。世界寬 3400，配額 11。開場提示星化變身教學
  （同系星彈滿 3 發，地面長按吸入鍵 0.6 秒）。
- **路線**：主線地面推進。單向平台 ×3（x≈700／1650／2680）、移動平台（x≈2150）、
  彈簧（x≈1150）、破磚 ×2（x≈800 彈藥／2300 回血——磁場干擾彈道時的救急補給）。
- **怪物**：magno 磁極獸 26% 主場（緩行 → 前搖預警圈 0.7 秒 → 磁場 1.9 秒：150px 磁域
  **吸偏飛行中的星彈且磁殼免傷星彈**——磁場期別浪費子彈，等磁場結束或改下砸／接觸吸入；
  恆可吸→雷鏈味）、zappy 16%、jelly 16%、glowy 16%、drilly 13%、spiky 13%。
