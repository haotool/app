# 地震知識小學堂 (Earthquake Kids Studio)

一個專為行動裝置設計的地震衛教網頁。

## 核心功能

- **分級教學**：提供幼兒園版、中階版與專業速查表。
- **互動模擬**：透過 SVG 動畫即時呈現地震規模 (Magnitude) 的視覺感受。
- **互動測驗**：5 題精選考題，幫助鞏固正確觀念。
- **高品質 UI**：使用 Tailwind CSS 與 Framer Motion 打造現代化視覺效果。

## 本地開發

1. 安裝依賴：`npm install`
2. 啟動開發伺服器：`npm run dev`
3. 建立正式版本：`npm run build`

## 技術棧

- React 18+
- TypeScript
- Tailwind CSS
- Framer Motion
- Inline SVGs
