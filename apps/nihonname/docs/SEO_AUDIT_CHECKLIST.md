1. 確保所有重要頁面回應 200
2. 設定正確的 robots.txt
3. 建立並提交 XML sitemap
4. 檢查 canonical 標籤
5. 確認 Search Console 設定

### Phase 2: P0 架構與內鏈（可走、可理解）

1. 優化 URL 結構
2. 建立麵包屑導航
3. 消除孤兒頁面
4. 設計內鏈策略

### Phase 3: P0 意圖對齊的內容重寫（可回答）

1. 每頁確保有清晰的 H1/H2 結構
2. 優化 Title 和 Meta Description
3. 確保內容匹配搜尋意圖
4. 加入 FAQ 區塊

### Phase 4: P1 CWV 與頁體驗（可用、可快）

1. 優化 LCP、INP、CLS
2. 圖片壓縮與 lazy loading
3. 減少阻塞資源
4. 設定 CDN 與快取

### Phase 5: P1 結構化資料（可展示）

1. 確保所有 JSON-LD schemas 完整
